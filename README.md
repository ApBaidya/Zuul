# Zuul
Text based adventure game

controls:
help = show controls
go --> (dir) = move room
drop --> (item) = drop item
take --> (item) = take item

Win condition:
[drop sword] at room [Heart] before opening [To The Isle]
Lose condition:
Forget to leave the sword and drown in your regrets haha.
